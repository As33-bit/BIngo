# Mahjong Bingo Game

這是一個麻將賓果試玩網頁，使用純 HTML/CSS/JavaScript 製作。

## 專案內容

- `index.html`：遊戲主程式
- `board.png`：6x6 背板
- `finger.png`：瞇牌手指圖
- `gmae.mp3`：背景音樂
- `55.png`：玩法介紹圖片
- `tiles_generated_aligned/`：麻將牌圖片素材

## 本機試玩

直接開啟 `index.html` 即可試玩。

也可以用本機伺服器：

```bash
python -m http.server 5175
```

然後打開：

```text
http://127.0.0.1:5175/index.html
```
